### Local Testing
```bash
cd /home/user/40thBdayAppRebuild
streamlit run app.py
```

Then open: **http://localhost:8501**

### With Real Weather
1. Get free API key from [OpenWeather](https://openweathermap.org/api)
2. Add to `.env`:
   ```
   OPENWEATHER_API_KEY=your_key_here
   ```
3. Restart app
